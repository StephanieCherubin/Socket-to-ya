# Socket-to-ya
Socket-to-Ya is a Javascript chat implemented with socket.io

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install Socket-to-Ya.

```bash
npm install
```

## Run

Run the server.js file as indicated in the package.json file.

```bash
node server.js
```
